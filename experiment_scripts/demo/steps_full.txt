Demo Steps:

1. Install Layer 4.5 on client and server 

2. Start NCO on client machine

3. Start DCA on client then server

4. Start Middlebox DCA on server

5. Deploy DNS front customization to server
   with inverse module to middlebox
   
6. Verify Challenge/Response 

7. Deploy DNS front customization to client

8. Verify Challenge/Response 

9. Start middlebox collection

10. Send DNS traffic

11. Revoke front customization modules

12. Deploy DNS end customization to client only

13. Send DNS traffic

14. Cleanup

15. Verify middlebox inverse
